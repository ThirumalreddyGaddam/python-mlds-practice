### String Operations
1. **String Repeater** - Repeats a word n times based on user input
2. **Character Counter** - Counts the number of characters in a string
3. **Case Converter** - Converts text to uppercase, lowercase, and title case
4. **First & Last Characters** - Extracts first and last characters from a word
5. **String Reverser** - Reverses a given string

### List Operations
6. **Range to List** - Creates lists using the range() function
7. **String to List** - Converts strings to character lists
8. **List Indexing** - Accessing elements by index (positive and negative)
9. **List Slicing** - Reversing lists and extracting specific portions
10. **List Append** - Adding elements to existing lists

- **Find Minimum of Three Numbers:** Functionality to return the smallest of three numbers.
- **Reverse a String:** Functionality to reverse any given string.
- **Check Vowel or Consonant:** Determine if a character is a vowel or consonant.
- **Factorial Method:** Calculate the factorial of a non-negative integer.
- **Interactive Calculator:** A calculator that continues to run until the user chooses to quit.
